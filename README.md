# Restaurant QR Ordering System - Frontend

React frontend for a QR-based restaurant ordering system.

## Features

- Browse menu items
- Scan QR code to order
- Shopping cart
- Place orders
- Kitchen Dashboard
- Admin Dashboard
- Real-time order status updates
- Responsive UI

## Backend Repository

This frontend communicates with the Spring Boot backend.

Backend Repository:
[RealTime-KOT-System-Frontend](https://github.com/yuvanprasath24/RealTime-KOT-System-Frontend)

## Tech Stack

- React
- Vite
- Axios
- React Router
- CSS
- WebSocket (STOMP)

## Running the Project

1. Clone the repository

```bash
git clone <https://github.com/yuvanprasath24/RealTime-KOT-System-Frontend.git>
```

2. Install dependencies

```bash
npm install
```

3. Configure the backend URL (Port : 8080).

4. Start the development server

```bash
npm run dev
```
